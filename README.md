## Tentang Project

Project ini Digunakan untuk Portfolio dan sebagai media blog untuk curahan hati saya, ya meski teknologinya gk keren keren amat, gk pke SOLID, atau apa pun itu saya sangat bangga bisa buat aplikasi ini.

## Install

- Type " composer install "
- Add .env file or rename .env.example to " .env "
- Type " php artisan key:generate "
- Type " npm i && npm run build "
